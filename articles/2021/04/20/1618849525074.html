1.计算整数列表的平均值

【问题描述】

给出一个整数列表，计算所有元素的平均值。

【输入形式】

输入列表，包括逗号，元素应逗号分隔

【输出形式】

小数，结果保留两位。

【样例输入】

[1,2,3,4,5]

【样例输出】

3.00

【样例说明】

用列表的形式输入—串整数，然后计算平均值，输出3.00，结果保留了两位小数。

【解题代码】

#计算整数列表平均值
a = eval(input())
suma = 0
for i in a:
    suma = suma + i
avg = suma / len(a)
print("%.2f" % avg)

2.删除列表中的最大元素和最小元素

【问题描述】

读入一个列表，删除其中的最大元素和最小元素，输出列表。最大元素和最小元素可能有多个。

【输入形式】

输入列表，包括方括号，元素应逗号分隔

【输出形式】

直接用print输出列表

【样例输入】

[1,2,3,4,5,6,1,7,7]

【样例输出】

[2,3,4,5,6]

【解答代码】

#删除列表最大最小值
a = eval(input())
mina = min(a)
maxa = max(a)
while mina in a:
    a.remove(mina)
while maxa in a:
    a.remove(maxa)
print(a)

3.计算圆周率近似值

【问题描述】

已知可以用下列公式计算圆周率的近似值。给定一个的近似值e，编程利用下列公式求得最接近e并且小于e的近似值，以及迭代次数n。

【输入形式】

从控制台输入e(e大于2，小于3.1415925)的值

【输出形式】

输出最接近e并且小于e的近似值，以及迭代次数n(以一个空格分隔，并且输出时要求小数点后保留7位有效数字)。

【样例输入】

3.14159

【样例输出】

3.1415896 18

【样例说明】

输入的的近似值e位3.14159，当n为18时计算的值为3.1415896，小于给定的精度值。当n为19时，计算的值为3.1415912，大于给定的精度值，所以利用上述公式求得的最接近e并且小于e的的近似值为3.1415896，对应的迭代次数为18次。

【解题代码】

#圆周率
e = float(input())
pia = 1.0
n = 2
k = 1.0
t = 1.0
while pia < (e / 2):
    pi = pia
    k = k * (n - 1)
    t = t * (2 * n - 1)
    pia = pia + k / t
    n = n + 1
pi = pi * 2
print("%.7f" % pi, end=' ')
print(n - 2)
